# tusinhentyo
通信専攻科課題の変調をする課題のソースコード。  
依存関係を減らすためにjavaで書いています。

プロジェクト自体はintellijで管理しているのでintellijで起動すると必要ライブラリなどを自動でダウンロードしてきてくれます。  


# 実行ファイル
[こちら](https://github.com/Khromium/tusinhentyo/releases/)にあります
